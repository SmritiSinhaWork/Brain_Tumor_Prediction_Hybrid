# Brain Tumor Detection Using Hybrid Model

An AI-powered web application for detecting brain tumors from MRI scan images using a Hybrid Deep Learning Model that combines CNN, Dynamic Mode Decomposition (DMD), and Vision Transformer (ViT).

## Features

- Upload MRI brain scan images
- Automatic tumor classification
- Real-time prediction results
- User-friendly web interface
- DMD-based feature enhancement
- Hybrid CNN + Vision Transformer model

## Tumor Classes

- Glioma
- Meningioma
- Pituitary Tumor
- No Tumor
